## vimrc

我的 Vim 配置文件。

这是一份私人配置，符合我的习惯，但并不照顾别人（你）的感受，如果有兴趣，参考一下就好。

## 安装

```sh
git clone https://github.com/zcodes/vimrc ~/.vim
cd ~/.vim && sh bootstrap.sh
```
